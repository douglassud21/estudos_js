1 - Estrutura Léxica

É a sintaxe de nível mais baixo de uma linguagem.

2 - Conjunto de caracteres

Os programas javascript são escritos com o conjunto de carcteres Unicode.

3 - Diferenciação de maiúsculas e minúsculas.

JavaScript é uma linguagem que diferencia lestras Maiúsculas de Minúsculas

Devem ser sempre Minúsculas no JavaScript

4 - Espaços em branco, quebras de linha e carcteres de controle de formato

O JavaScript ignora os espaços que aparecem entre sinais em programas.

Também ignora quebra de linha.

5 - Sequência de escape Unicode

Alguns hardwares não possuem capacidade de serem introduzidos livremente em algumas máquinas. Por conta disso, o JavaScript utiliza uma técnica para ajudar esses hardwares mais antigos a se comunicarem.

6 - Normalização

O padrão Unicode define a codificação preferida para todos os caracteres e especifica um procedimento de normalização para converter texto em forma canônica conveniente para comparações.

javascript presume que o código-fonte que está interpretando já foi normalizado e não tenta normalizar indentificadores, string nem expressões regulares.

7 - Comentários

// <= Esse é um comentário de uma linha.

/* <= Assim também.

/**/ <= Esse e um comentário de varias linhas.

8 - Literais

Um literal é um valor de dados que aparece diretamente em um programa.

9 - Indentificadores e palavras reservadas

Um indentificador é simplesmente um nome.

10 - Palavras reservadas

Javascript reserva vários identificadores como palavras-chave da própria linguagem, como:

break | case | catch | continue | debugger | default | delete | do | else | false | finally | for | function | if | in | instanceof | new | null | return | swith | this | throw | true | try | typeof | var | void | while | with

Podem passar a ser:

class | const | enum | export | extends | import | super ]

Reservadas no modo restrito:

implements | interface | let | package | private | protected | public | static | yield
