Tipos

Em JavaScript, os tipos de dados, também conhecidos como tipos, definem o tipo de valor que uma variável pode armazenar. São a base para a manipulação de informações em JavaScript, e incluem tipos primitivos e tipos de objeto.

Em JavaScript, os tipos primitivos são valores imutáveis que não possuem métodos e que são armazenados por valor em memória. Existem sete tipos primitivos:
String, Number, Boolean, Null, Undefined, Symbol e BigInt.

    primitivos

  • String: Representa sequências de caracteres, como "Olá, mundo!".
  • Number: Representa valores numéricos, como 10, 3.14 ou -2.5.
  • Boolean: Representa valores lógicos, True ou False.
  • Null: Representa a ausência de valor, é um valor especial que indica uma variável que foi definida, mas não tem um valor associado.
  • Undefined: Representa uma variável que foi declarada, mas ainda não tem um valor atribuído.
  • Symbol: Representa um valor único e imutável, frequentemente usado como chaves de objetos.
  • BigInt: Representa valores inteiros com precisão arbitrária, útil para trabalhar com números muito grandes.

    Objetos

  • Array: Uma lista ordenada de valores.
  • Object: Uma coleção de pares chave-valor, base do JavaScript.
  • Function: Um bloco de código reutilizável.
  • Date: Representa datas e horas.
  • RegExp: Representa expressões regulares.
  • String: Objetos que envolvem tipos primitivos string.
  • Math: Um objeto que contém constantes e funções matemáticas.
  • JSON: Um objeto que permite a manipulação de dados em formato JSON.

Coisas importantes

Qualquer valor que não seja:

Número, String, boleano, Null e undefined é um Objeto
